#### In case you're having problems with OpenSSH in Windows:
- Navigate to ~/.ssh and create an extension-less file called "_config_"
- Insert the following into the file, save, and retry the operation that didn't work:
```
Host gitlab.com
	UpdateHostKeys no
```
